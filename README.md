# Quiz App Backend

This is the **.NET 8 backend** for the Quiz App project.

## Tech Stack
- .NET 8 / ASP.NET Core Web API
- C#  
- xUnit & Moq for testing

## Project Structure
- `QuizApp.Backend` → API project
- `QuizApp.Tests` → Unit tests for services and controllers
- `QuizBackend.sln` → Solution file

## Run Locally
```bash
cd QuizApp.Backend
dotnet restore
dotnet build
dotnet run

## Run Tests:
cd QuizApp.Tests
dotnet test
